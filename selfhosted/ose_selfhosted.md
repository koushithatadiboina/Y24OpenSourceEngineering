# Self Hosted Project - BOOKSTACK

## Description
**BookStack** is an open-source, self-hosted platform for storing and organizing information like documentation, wikis, or notes.  
Itâ€™s built with PHP and Laravel and offers a clean, user-friendly interface for creating pages, organizing them into chapters and books, and managing access for teams.  
BookStack is ideal for knowledge bases, internal documentation, or collaborative writing.


## Installation 
To install **BookStack**, you need a system with PHP, Composer, and MySQL (or MariaDB).  
1. Download or clone the BookStack repository from GitHub.  
2. Set up a database and connect it in the `.env` file.  
3. Install dependencies using Composer.  
4. Run the database migrations and start the local server.
